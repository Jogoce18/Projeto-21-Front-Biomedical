# Description
<p align="justify">
<b>Biome Optimus </b> is an application for the control of preventive and corrective maintenance of medical equipment in companies that provide this service.

</p>

</br>

## Features


-   Create new maintenance
-   Get all maintenance
-   Create new client
-   Get clients
-   Create new observations 
-   Get observations
-   Edit maintenance                                                                               

</br>
